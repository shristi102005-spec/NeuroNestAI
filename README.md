# NeuroNest AI

NeuroNest AI is a GenAI-powered study assistant built using Python, Streamlit, LangChain, and OpenRouter APIs.

## Features

- AI Chatbot
- PDF Upload & Processing
- RAG-based Question Answering
- Vector Database using ChromaDB
- AI Quiz Generator
- AI Notes Generator
- Chat Memory
- Semantic Search

## Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- OpenRouter API
- PyMuPDF

## How It Works

1. Upload PDF
2. Extract text
3. Create embeddings
4. Store in vector DB
5. Retrieve relevant chunks
6. Generate AI responses

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
